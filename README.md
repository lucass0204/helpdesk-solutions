## 📌 Sobre o projeto

Este repositório foi criado com o objetivo de demonstrar, na prática, habilidades em suporte técnico de TI (Helpdesk), por meio da documentação de problemas reais e simulados do dia a dia.

Aqui são registrados processos de diagnóstico, análise e resolução de falhas comuns enfrentadas por usuários, com foco em:

* Conectividade de rede (TCP/IP, DNS, DHCP)
* Problemas em sistemas Windows
* Troubleshooting passo a passo
* Automação de tarefas com scripts simples

Cada caso é documentado como um chamado técnico, contendo descrição do problema, diagnóstico, possível causa, solução aplicada e resultado final.

## 🎯 Objetivo profissional

Este projeto faz parte do desenvolvimento prático na área de suporte técnico, visando consolidar conhecimentos adquiridos em cursos como AWS Cloud Practitioner, Cisco Networking Basics e Google IT Support, além de demonstrar capacidade de resolução de problemas, organização e documentação técnica.

## 🚀 Diferencial

Ao invés de apenas teoria, este repositório apresenta aplicações práticas e estruturadas, simulando situações reais de atendimento em ambiente de suporte técnico.
